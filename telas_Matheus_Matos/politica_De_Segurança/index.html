Política de Segurança:

Este documento explica como o Solidarize-se trata os dados coletados em todas as frentes.
Política de Privacidade - Projeto Solidarize-se
Última atualização: 02 de abril de 2026.

1. Informações Coletadas Coletamos informações necessárias para a segurança da plataforma e operação das atividades sociais, incluindo:

  • Dados Cadastrais: Nome completo, e-mail, RG, CPF e telefone para identificação de voluntários e parceiros.
  • Dados de Localização: Endereços para o cadastramento de pontos de coleta e logística de doações.
  • Registros de Acesso: IP e horários de conexão para fins de segurança e validação de conta.

2. Uso dos Dados Os dados coletados são utilizados exclusivamente para:

  • Viabilizar o contato entre a ONG e voluntários.
  • Exibir pontos de coleta no mapa interativo para os usuários.
  • Garantir a segurança dos usuários, evitando cadastros falsos ou mal-intencionados.

3. Compartilhamento de Informações O Solidarize-se não vende ou aluga dados pessoais. As informações de pontos de coleta (endereço e nome do local) são públicas para que doadores possam encontrá-los. Dados sensíveis de voluntários são mantidos em sigilo administrativo.

4. Segurança Implementamos medidas técnicas de proteção para garantir que seus dados não sejam acessados por terceiros não autorizados.
